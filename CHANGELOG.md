# Change Log

## [0.1.0] - 2026-06-25

### Added
- Initial release
- Multi-language scanner supporting 20+ file extensions
- Configurable tags: TODO, FIXME, HACK, XXX, BUG, NOTE, OPTIMIZE, REVIEW
- `Scan Workspace` command — full codebase scan with output panel
- `Scan Current File` command — quick active-file scan
- `Export as Markdown` command — formatted report for sharing
- `Export as JSON` command — machine-readable for CI/CD integration
- Author detection from `TODO(name)` syntax
- 36 unit tests covering scanner logic (100% pass rate)
